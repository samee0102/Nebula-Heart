## Cosmic Nebula Heart

Cosmic Nebula Heart is an interactive 3D heart-shaped nebula made with Three.js, particles, and bloom effects. Touch or click to explode the particles and watch them slowly reform into a glowing cosmic heart.

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES modules)  
- Three.js with OrbitControls, EffectComposer, and UnrealBloomPass  

## Files

- `index.html` – Base HTML file, import map, and script loading  
- `style.css` – Fullscreen layout and UI styling  
- `main.js` – Three.js scene, particles, interactions, and animation loop  

## How to Run

1. Clone this repository.  
2. Use a local server (for example: VS Code Live Server, `npx serve`, or any static server).  
3. Open `index.html` in the browser through the local server URL.
